Oh! My Termux

发表于 2018-08-24 | 更新于 2018-09-08 | 分类于 教程 | 评论数： | 热度： ℃
本文字数： 7.1k | 阅读时长 ≈ 6 分钟

Termux 是个非常强大的终端，它有普通终端的功能又能运行部分 Linux 软件，如 nodejs，clang，python，nginx 等等。并且能通过 PRoot 实现无需 root 在 Android 上运行 Linux（arm）。

初始化

Termux 可在各大应用商店下载，装好后一次打开需要连网下载一些东西（需科学上网）。加载完成后即可使用了。Termux 初始话完成后可以看见已经装好了一些软件包。这些软件包都是 Termux 必须的包一定不要卸载了，不然就准备重新加载吧。（ps:如果没办法科学上网可以去用 这个和 Termux 差不多）

换源

Termux 默认源是国外的，这就是为什么第一次打开需要科学上网的原因。初始话完成后就可以随意使用了，但如果要安装一些其他软件时还是要科学上网，不然要么没速度要么 404 。好在有 Termux 的镜像源。我用的是 tuna 的源速度还可以。
如果有 root 权限可直接去 Termux 文件路径（/data/data/com.termux/files/usr/etc/apt) 更改 sources.list,没有的话可以在 Termux 内 输入 apt edit-sources 如果出现 E: Sub-process editor returned an error code (100) ，则需要设置一下$EDITOR: export EDITOR=vi && apt edit-sources 按 i 进行修改，改好后按 Esc 输入 :wq 回车即可。（在屏幕左上角右滑会出现一个界面,按住 KEYBOARD 和 音量 - 即可唤出底部键盘。附 Termux 快捷键介绍）

安装 ssh

pkg in openssh 安装 openssh，装好后即可使用 ssh 连接 Linux 设备。也可使用其他设备连接 Termux，不过 Termux 只能使用秘钥连接。在客户端输入 ssh-keygen -t rsa 生成秘钥，运行将 .ssh 目录下的 id_rsa.pub 文件复制到服务端也就是 Termux 主目录下的 .ssh 文件夹。运行 cat id_rsa.pub >>authorized_keys 之后运行 sshd 即可，以后启动直接运行 sshd 就行了。启动 sshd 后 使用 whoami 查看当前用户，ssh ua_axxx@xxxxx -p 8021 就能连接了。ua_axxx 为前面 whoami 得到的，@ 后面为当前设备 ip ，不知道的可以运行 ifconfig 查看。

安装配置 ohmyzsh

Termux 默认的是 bash ，切换为 zsh 有配置好的脚本可以直接使用termux-ohmyzsh 。运行 sh -c "$(curl -fsSL https://github.com/Cabbagec/termux-ohmyzsh/raw/master/install.sh)" 等待一下就好了（中间会同步 github ohmyzsh 项目，建议使用科学上网）。颜色和字体我选的是第 0 项和第 6 项，随自己喜好选择。输入 zsh 运行或者重启 Termux 即可使用 zsh。接下来是配置 zsh 。我介绍下我自己的配置。

  1. ZSH_THEME="xiong-chiamiov-plus" 主题，可在 .ohmyzsh/themes 中找，也可下载主题文件放入此目录（注意文件权限）

  2. plugins=(git z zsh-autosuggestions zsh-syntax-highlighting extract)

    zsh 插件，可在 .ohmyzsh/plugins 中找。其中 zsh-autosuggestions 、zsh-syntax-highlighting 需自行安装。

    • git: 提供一些 git 的快捷命令
    • z: 文件目录跳转，不同的是可以不用目录路径及全名，需用过 cd 命令到目录。
    • zsh-autosuggestions: 命令提示插件
    • zsh-syntax-highlighting: 高亮插件

    • extract: zsh 自带解压插件，用法为 x+压缩文件，需已安装对应解压缩软件。
      同步 zsh-autosuggestions 以及 zsh-syntax-highlighting 命令。

  3. 别名，这个可以为命令取个别名，减少输入量。

安装配置 aria2

aria2 是一款多线程下载器，能下载磁力和种子。输入 apt in aria2进行安装。使用 touch aria2.conf 创建配置文件，touch aria2.session 创建 aria2 状态文件。aria2 配置文件可在网上找或直接使用我的。

使用命令 aria2c --conf-path=aria2.conf 启动，不过这样会导致终端用不了，只能重新创建个窗口，而且当终端关闭后 aria2 也就关闭了。网上说可以使用 -D 参数解决，但我测试这样无法启动 aria2。浏览器打开 <aria2c.com> 即可使用。也可使用 AriaNg。（下载 AriaNg 压缩包 wget -O ./storage/downloads/master.zip https://github.com/mayswind/AriaNg-DailyBuild/archive/master.zip,之后用文件管理器去内置存储的 Download 文件夹解压，用浏览器打开 index.html 就行了。）

安装配置 vim

Vim 是一个类似于 Vi 的著名的功能强大、高度可定制的文本编辑器。我也才接触 vim 所以没什么能说的直接贴配置吧。
使用 vim-plug 管理插件

  1. 语法高亮，检查

2.自动规范化中文排版
Plug 'hotoo/pangu.vim'

哔哩哔哩直播挂机

BilibiliHelper
这个很简单，装好 PHP 填写配置直接运行就可以了。

1.克隆项目

git clone https://github.com/metowolf/BilibiliHelper.git

2.修改配置

cd BilibiliHelper && vim config

3.安装 PHP

pkg in  php

4.运行

php index.php

上使用 BaiduPCSD-Go 高速下载百度云文件。
